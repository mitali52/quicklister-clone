# Feature Inventory

Complete list of features observed on quicklister.co.uk.

---

## Listing Management

| Feature | Detail |
|---|---|
| Create listing | Supports residential sale, residential let, commercial sale, commercial let |
| Edit listing | Update description, price, and details at any time while active |
| Drag-and-drop photo upload | Upload and reorder photos via drag-and-drop UI |
| Photo reel | Ordered photo gallery created from uploaded images |
| Floorplan upload | Attach floorplan as a document to the listing |
| Document/certificate upload | Gas safety, EPC, electrical certificates, etc. |
| Pause listing | Temporarily deactivate a live listing without deleting it |
| Re-list / resume | Re-activate a previously paused listing |
| Remove listing | Delete a listing with a few clicks |
| Mark as Sold / Let | Archive a completed listing |

---

## Portal Syndication

| Feature | Detail |
|---|---|
| Rightmove syndication | Available in Premium sales package and as optional add-on |
| Zoopla syndication | Included in all packages |
| OnTheMarket syndication | Included in Exposure and above packages |
| PrimeLocation syndication | Included in selected packages |
| 24-hour go-live SLA | Listings go live on portals within ~24 hours of approval |
| Portal status tracking | Platform shows syndication status per portal |

---

## Identity & Verification

| Feature | Detail |
|---|---|
| Photo ID upload | Driving licence, passport, visa, or residence permit |
| Proof of ownership upload | Land registry, mortgage statement, property deeds, solicitor letter, buildings insurance, service charge statement |
| KYC check | Address validation and electoral roll validation |
| AML (Anti-Money Laundering) check | Optional add-on — £29.99; verifies buyer/seller identity |
| Admin review and approval | Quicklister staff manually approve ID and ownership docs before listing goes live |

## Communication & Enquiry Management

| Feature | Detail |
|---|---|
| Centralised inbox | All buyer/tenant enquiries in one place |
| Email notifications | New enquiry alerts sent to seller/landlord by email |
| 24/7 enquiry notifications | Available on Exposure and Premium plans |
| Office-hours enquiry notifications | Saver plan (sales) — 9am–6pm Mon–Fri |
| Real-time / web chat | Direct messaging with buyers or tenants inside the platform |
| Enquiry from portal | Enquiries submitted on Rightmove/Zoopla are routed to the platform inbox |
