## Pricing Logic Rules

1. **One package per listing** — each listed property requires its own package purchase (except Pro Lister subscription which covers up to 6/month).

2. **Rightmove is premium-only by default** — it is included in the Premium sales tier and the Pro Lister subscription. For Saver and Exposure sales packages, Rightmove is an optional add-on at £159.

3. **Lettings packages are cheaper and shorter** — 6-week lettings vs 3-month sales; reflects faster market turnover.

4. **Pro Lister is subscription-based** — designed for landlords with multiple properties. Monthly billing with a 3-month minimum lock-in. Yearly billed upfront.

5. **Tenant referencing is pay-per-use** — charged per applicant regardless of the listing package tier.

6. **AML check is optional for all** — sellers/landlords can elect to run it; may become effectively required depending on buyer/solicitor expectations.

7. **No commission on sale/let proceeds** — Quicklister earns only through upfront package fees and add-on fees. There is no percentage commission on the transaction value.
